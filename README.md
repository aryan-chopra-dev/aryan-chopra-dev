## 🚀 Featured Projects

### 🎬 YouTube Transcript RAG Assistant
> *Semantic Video Q&A System — Production Deployed*

A full-stack RAG system that lets you **ask questions about any YouTube video** using its transcript.

- **Stack:** FastAPI · Groq (Llama-3.3-70B) · FAISS · HuggingFace Embeddings · Streamlit · Chrome Extension
- **Highlights:** 512-token chunking · 384-D embeddings · <500ms retrieval latency · 95%+ accuracy on 10,000+ token transcripts
- **Interfaces:** Web App + Chrome Extension with real-time video detection
