- Implemented:

  - **Embeddings (384-dim MiniLM)**

- Achieved:
  - ⚡ **<500ms retrieval latency**
  - 🎯 **High answer relevance & accuracy**

- Developed **Chrome Extension + Web UI** for real-time interaction  
- Designed as a **scalable API-driven backend system**

- Developed **FastAPI backend (REST APIs)** for real-time querying
- Containerized using **Docker** and deployed on cloud
- Focused on **latency optimization & scalable system design**
